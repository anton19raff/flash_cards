# World Greetings Trainer

Веб-приложение на Flask для изучения слов приветствия на 50 языках мира.

## Автор
- Студент: **Антон Рафф**
- Группа: **М01-510а**

## Описание проекта
Приложение относится к образовательной тематике и помогает изучать базовые приветствия.

Функции:
- каталог карточек по языкам;
- страница деталей карточки;
- добавление/редактирование/удаление карточек;
- мини-квиз для тренировки;
- страница статистики.

## Соответствие формальным критериям
- Минимум 4 страницы: `home`, `greetings`, `greeting_detail`, `greeting_form`, `quiz`, `stats`.
- Минимум 2 формы: форма карточки (добавление/редактирование), форма ответа в квизе.
- Минимум 2 страницы вывода данных: список карточек + детали карточки (+ статистика).
- Валидация форм с читаемыми сообщениями об ошибках.
- HTML-шаблоны с блоками и наследованием (`templates/base.html`).
- Собственные стили (`static/style.css`).
- Хранение данных в SQLite (`greetings.db`).

## Стек
- Python 3.10+
- Flask
- SQLite
- HTML/CSS
